# ZoomRunner

Simple utility which automates joining a Zoom meeting with multiple sandboxed Zoom instances.

## Usage
- Install [Sandboxie](https://github.com/sandboxie-plus/Sandboxie)
- Create multiple boxes with names:
    - `SB1`
    - `SB2`
    - `SB3`
    - ...
- Open the executable
- Fill in the values:
    - `Zoom Link` &ndash; Zoom meeting URL
    - `Number of Zooms` &ndash; Number of Zoom instances to be opened
    - `Main Zoom NOT sandboxed` &ndash; If checked, the first Zoom instance will not be sandboxed
- Click `Join`

The tool automatically closes and the Zoom meeting windows will open in a few seconds.
